# Task-Manager
A Windows desktop application built to manage daily tasks. This project demonstrates desktop UI development, local data persistence using JSON, and time-based logic for task expiration.

# Features
Task Management: Create, read, update, and delete (CRUD) tasks.

Local Storage: Tasks are serialized and saved locally to a .json file, ensuring data persists between application restarts.

Expiration Tracking: Built-in logic to check task due dates and flag/highlight expired tasks.

# Technologies Used
Language: C#

Framework: .NET (Windows Forms)

IDE: Visual Studio 2022

Data Storage: JSON (Local File System)

# How to Run
Clone this repository to your local machine.

Open the .sln (Solution) file in Visual Studio 2022.

Click "Start" (or press F5) to build and run the application.
